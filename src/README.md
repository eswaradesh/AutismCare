## 🌟 Welcome to the `src` Directory

This directory contains the core frontend source code of our application. We focus on delivering a high-quality, emotionally resonant user experience for parents, caregivers, and therapists. The interface is completely modern, adopting **dark-mode styling, glassmorphism**, and fluid interactions to make handling daily routines as comfortable as possible.

## 🏗️ Folder Structure

```text
src/
├── components/   # Reusable UI components (buttons, cards, forms)
├── contexts/     # React Context for global state (Auth, Theme)
├── hooks/        # Custom React hooks for business logic 
├── integrations/ # API and Third-party integrations
├── lib/          # Utilities, formatters, and helpers
├── pages/        # Main application views and route components
├── types/        # Global TypeScript interfaces
├── App.tsx       # Main Router and Theme Provider
└── main.tsx      # Entry Point of the React tree
```

## ✨ Design Philosophy

1. **Accessibility First**: Color-contrast checked and screen-reader friendly.
2. **Modular Components**: Built heavily with specialized React constructs for maximizing code reusability.
3. **Optimistic Updates**: Immediate feedback upon user actions before server responses.
4. **Therapeutic Aesthetics**: A calm, focused palette utilizing subtle micro-animations to prevent sensory overload.

## 🚀 Development Mode

The project heavily relies on **Vite** for incredibly fast Hot Module Replacement (HMR).

To spin up the web app locally:

```bash
# From the root directory:
npm install
npm run dev
```
Access the application at [http://localhost:5173/](http://localhost:5173/) or your indicated port.

## 🧩 Key Libraries in use

- **React Router DOM**: Client-side routing with complex nested layouts.
- **Tailwind CSS**: Instant utility-class styling without leaving TSX.
- **Lucide React**: Crisp, modern SVG icons.
